# Financial Transactions Power BI Dashboard

## Overview
This project analyzes financial transactions using Power BI.

## Key Features
- Star Schema Data Model
- 25+ DAX Measures
- 5 Dashboard Pages
- Customer, Account, Product & Time Analysis

## Tools Used
- Power BI
- DAX
- Data Modeling

## Insights
- Identified high-value customers
- Analyzed transaction trends
- Compared deposits vs withdrawals

## Files
- PBIX Dashboard
- Dataset
- Screenshots

## How to Use
1. Download PBIX file
2. Open in Power BI Desktop
3. Explore dashboards
